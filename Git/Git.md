# Git usage

- Set parameters

```bash
git config --global.user.name "First Last"
git config --global user.email "email@domain.com"
```

- clone
```bash
git clone <url>
```

- Directory will be in terminal now

## Git Status

- Up to date with origin/main

```bash
git status

cat .git/config
```

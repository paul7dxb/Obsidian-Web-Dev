# Selectors

```CSS
/*element*/
button{
	property: value;
}

/*class*/
.className{
}

/*ID*/
#eleID{
}

/*nested*/
parentEle childEle{
}

/*Multiple selectors*/
button, p{
}
```

# Inheritance

For elements that are not inheriting desired property
```CSS
button{
	color: inherit;
}
```

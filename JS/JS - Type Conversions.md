# JS Int Rounding

```JS
// Floor
console.log(Math.floor(5.95));    // Expected output: 5

// Ceiling
console.log(Math.ceil(.95));    // Expected output: 1

// Rounding
console.log(Math.round(5.95), Math.round(5.5), Math.round(5.05));
// Expected output: 6 6 5
```

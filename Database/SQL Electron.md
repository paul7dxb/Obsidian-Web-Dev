# Connect to AWS DB

- Get database endpoint from AWS console
- In SQL Electron
	- Add
	- Input a name
	- Select DB type
	- Paste host into server address
	- Enter user/pass from setup
	- Enter db name from setup
	- Click Test
		- If failed check:
		- Public access turned on
		- Security groups may require specific IP
	- Save
	- Connect

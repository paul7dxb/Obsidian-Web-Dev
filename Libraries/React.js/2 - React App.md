Take an app from prototype to production

Includes:
- react
- React scripts
	- Babel
- react-dom
- File Bundling

# Set up

Install Node.js
https://nodejs.org/en/

Check Version:
```bash
//Browse to project folder

node -v
npm -v
npx create-react-app react-app
cd react-app
npm start

```

open on localhost:3000
